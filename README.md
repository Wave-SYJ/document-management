# 文献搜索和管理系统

软件工程导论大作业

## 主要技术栈

本项目为后端渲染的Web应用。

前端：SemanticUI + JQuery

模板引擎：Freemarker

后端：SpringBoot + MyBatis

数据库：MySQL
